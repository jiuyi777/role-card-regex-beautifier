# 角色卡正则美化助手

适用于 SillyTavern 1.14.0 及以上。

## 它做什么

插件会读取当前单角色聊天所使用的角色卡，把角色设定、性格、场景、原开场白、备用开场白和示例对话作为只读内容依据；再结合你导入的参考正则、启用的固定指令、风格预设和本次临时命令，调用酒馆当前已经连接的 AI，生成一个新的角色适配正则 JSON。

角色卡与原开场白不会被修改。生成结果默认只显示在预览区，必须由你点击确认后才会保存到当前角色正则。

## 入口

- 输入框左侧魔棒菜单：`角色卡正则美化`
- 斜杠命令：`/regex-beauty`

## 主要功能

- 自动读取当前角色卡
- 导入任意正则 JSON 作为参考美化母版
- 将当前角色已有正则保存为母版
- 保存多个参考母版并切换
- 添加、导入、导出可开关的固定指令条目
- 保存、导入和导出可选的风格预设
- 固定指令与风格预设分别管理、分别导入；选择预设后仍可继续增删固定指令
- 每次生成前输入临时命令
- 静默调用酒馆当前模型，不向聊天窗口发送消息
- 查看并手动编辑完整结果
- 复制或下载结果 JSON
- 确认后新增或覆盖当前角色正则
- 对联网、Cookie、iframe、eval 等高风险内容进行基础静态拦截

## 安装

将整个 `role-card-regex-beautifier` 文件夹放入：

`SillyTavern/public/scripts/extensions/third-party/`

然后重新加载 SillyTavern 页面，在扩展管理中确认“角色卡正则美化助手”已启用。

## 使用顺序

1. 打开一个单角色聊天。
2. 从魔棒菜单打开“角色卡正则美化”。
3. 导入一个参考正则 JSON，或把当前角色已有正则存为母版。
4. 添加或导入需要每次发送的固定指令。
5. 按需保存、导入或选择一套风格预设；不使用预设也可以生成。
6. 输入本次临时命令。
7. 点击“发送给 AI 并生成正则”。
8. 检查结果后复制、下载，或确认保存到当前角色。

## 当前限制

- 第一版只支持单角色聊天，不处理群聊中的成员选择。
- 生成质量取决于酒馆当前连接的模型、上下文长度和输出长度。
- 静态检查不能证明 AI 生成的 HTML、CSS 或 JavaScript 一定安全、正确或视觉可用，保存前仍需人工阅读。
- 本插件不会自动启动酒馆、不会自动打开浏览器，也不会自动启用生成结果。
